# Game Boy 2048 游戏说明书

## 游戏简介

2048 是一款经典的益智类游戏，目标是在 4x4 的网格中通过滑动方块，将相同数字的方块合并，最终得到 **2048** 方块。

本版本专为 **Game Boy / Game Boy Color** 开发，使用 GBDK-2020 工具链编写。

## 游戏规则

### 基本玩法

1. **开始游戏**：游戏开始时，网格上会随机生成两个方块（数字 2 或 4）

2. **移动方块**：按方向键使所有方块向该方向滑动
   - 当两个**相同数字**的方块碰撞时，它们会**合并**成一个新方块
   - 新方块的数字是原来两个方块数字的**和**

3. **生成新方块**：每次移动后，会在空位随机生成一个新方块（2 或 4）

4. **获胜条件**：当网格中出现 **2048** 方块时，游戏获胜

5. **失败条件**：当网格被填满且无法进行任何合并时，游戏结束

### 示例

**移动前：**
```
+---+---+---+---+
| 2 | . | . | 2 |
+---+---+---+---+
```

**向左移动后（两个 2 合并成 4）：**
```
+---+---+---+---+
| 4 | . | . | . |
+---+---+---+---+
```

**得分**：+4 分

---

## 计分规则

- 每次合并方块时，**新方块的数字**会计入总分
- 例如：合并两个 8 得到 16，得 **16 分**

---

## 策略提示

1. **保持一个角落**：尽量将最大的方块保持在某个角落，避免它被分散

2. **有序排列**：尝试让方块按大小顺序排列，便于连续合并

3. **避免小数字在大数字旁边**：这会阻碍合并

4. **谨慎使用重置**：按 A 键会立即重新开始游戏

## 编译说明

### 前置要求

1. 安装 **GBDK-2020**
   - 下载地址：https://github.com/gbdk-2020/gbdk-2020/releases
   - macOS: 解压到 `/opt/gbdk` 或自定义路径
   - Windows: 解压到 `C:\gbdk` 或自定义路径

2. 配置环境变量
   - macOS/Linux: `export PATH=/path/to/gbdk/bin:$PATH`
   - Windows: 添加 `C:\gbdk\bin` 到 PATH

### macOS / Linux 编译

```bash
cd gbdk-2048

# 编辑 Makefile，修改 GBDK 路径
# GBDK_HOME = /path/to/your/gbdk

# 编译
make

# 输出：2048.gb
```

### Windows 编译

```batch
cd gbdk-2048

REM 编辑 build.bat，修改 GBDK 路径
REM set GBDK_HOME=C:\gbdk

REM 编译
build.bat

REM 输出：2048.gb
```

### 手动编译

```bash
# macOS/Linux
/path/to/gbdk/bin/lcc -mgbz80:gb -Wl-yt0x1B -Wl-j -Wm-yoA -Wm-ya4 -autobank -Wb-ext=.rel -Wb-v -o 2048.gb src/main.c src/game.c
```

## 故障排除

### 问题：按键无反应
- **解决**：检查模拟器的按键映射设置
- SameBoy 默认：A=X, B=Z, Start=Enter

### 问题：画面显示异常
- **解决**：确保使用正确的 GBDK 版本
- 推荐使用 GBDK-2020 release v4.3.0 或更高版本

### 问题：编译失败
- **解决**：检查 GBDK 路径配置
- 确保 `lcc` 编译器在 PATH 中
